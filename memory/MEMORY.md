# Memory index — claude-goal starter pack

Skill-related operating memories (`type: feedback`), de-branded, that pre-load the goal-runner
discipline. One line per memory; see [docs/MEMORY.md](../docs/MEMORY.md) for the convention.

- [Goal means never-stop / never-ask](goal-means-never-stop-never-ask.md) — inside /goal, run fully autonomously; questions and turn-ends defeat the skill.
- [Never narrate during a goal](never-narrate-during-goal.md) — zero prose between tool calls; narration is a pause is a stop.
- [Build, don't survey](build-dont-survey.md) — trying+failing beats analyzing+asking; pick the highest-leverage path and execute.
- [Plateaus are progress](plateaus-are-progress.md) — a metric stuck flat is exploration; block only on the same approach failing twice; shift categories.
- [Failures are data, never "dead"](failures-are-data.md) — reframe outcomes as "config X refuted, next varies Z"; never close a door.
- [Verify before claiming done](verify-before-claiming.md) — "done/live" means actually running + exercised; self-test before believing anything.
- [Row counts aren't quality](row-counts-arent-quality.md) — exit-0 + count verify the pipeline, not the content; spot-check random records.
- [Diagnose, then implement](diagnose-then-implement.md) — the deliverable is the fix, not the report; don't bail early at a clean stopping point.
- [Rules as code, not memory](rules-as-code.md) — enforce load-bearing rules with hooks/guards/checks; prose alone gets forgotten.
- [Concurrent goals: the PAUSE brake](concurrent-goals-pause-brake.md) — multiple autonomous CLIs can lock a machine; `touch .claude/goals/PAUSE` halts all.
- ["All" means 100%](all-means-100-percent.md) — no cherry-picking; a found-but-unclosed gap is not complete; the coverage gate enforces it.
